Variáveis

Tipos de variáveis

  • String
  • number
  • boolean

Regras para delaração de variáveis

  • Não podem ser iniciadas com números, apenas com letras ou "_"
  • Não podem ser utilizados caracteres especiais como "ç", "^", "~"...
  • Não podem ser iguais as palavras reservadas da liguagem

Declarando variáveis com a instrução "var"

Case sensitive

  • var curso, var Curso e var CURSO são variáveis diferentes

Exemplos

  • Declarações válidas: var nome, var AulaVariaveis, var _idade
  • Declarações inválidas: var 123nome, var número
